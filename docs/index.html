AppDiet Tracker

Simple calorie tracking you can keep.

食事の細かい入力ではなく、カロリー記録に絞ったシンプルなダイエット記録アプリです。 簡単に記録し、知人とのグループで進捗を見える化し、Apple Health とも連携できます。

Core Value

  • Complicated meal logging is hard to sustain, so the app focuses on calories only.
  • Logging is designed to be quick and low-friction.
  • You can create groups with people you know and track achievement progress together.
  • Apple Health integration helps bring burned calorie data into the app.
